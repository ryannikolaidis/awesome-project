Installation
============

From Source
-----------

.. code-block:: bash

    git clone https://github.com/ryannikolaidis/awesome-project.git
    cd awesome-project
    make install

Development Installation
------------------------

.. code-block:: bash

    git clone https://github.com/ryannikolaidis/awesome-project.git
    cd awesome-project
    make install-dev

Requirements
------------

* Python 3.12+
